立创&涂鸦训练营第二期--物联网温湿度传感器
==
一款基于涂鸦智能的小温湿度传感器，SHT20传感器自动采集当前温湿度，通过STM32单片机&涂鸦wifi模块上传至云端，并在手机APP端显示采集的温湿度数据
--

## 整体介绍

* 本次工程是一个基于涂鸦智能的温湿度传感器，采用低代码 MCU 方案，即将涂鸦物联网模块与ST32F0单片机结合起来使用的方法，实现温湿度数据的采集和上报，将数据可视化。
